Alphabets and Languages

Alphabets

An alphabet is a set of finite symbols. An arbitrary alphabet is denoted with Σ

String

A string is a set of finite sequence of symbols from the alphabet

Formaly, a string w = w1w2...wn where each wi is a symbol in an alphabet Σ

Empty String

The empty string is the string with no symbols denoted ε or λ

The set of all strings, including ε, over an alphabet is denoted as Σ*

Length

The length of a string (also known as the cardinality) is the length as a sequence. If there exists a string w then the length of w is denoted |w|

The symbol in the ith position is denoted as wi

Operations and relations on strings

Concatenation

Given two strings w1 and w2, the concatenation of w1 and w2 would be the string w1w2

All concatenation is, is sticking two strings end-to-end

Also note, concatenation is associative, meaning if we have three strings xyz, we can combine as (xy)z or x(yz)

xn is the string obtained by concatenating the string x n times
